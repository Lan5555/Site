Examples to code

Coding challenges are a great way to improve your programming skills, keep your mind sharp, and learn new things. These challenges come in many different forms, from puzzles and games to programming problems and algorithmic challenges. In this article, we’ll explore five fun coding challenges that can help you sharpen your skills and have a great time doing it.

1. FizzBuzz
FizzBuzz is a classic coding challenge that is often used in job interviews to test a candidate’s ability to write simple code. The challenge is simple: write a program that prints the numbers from 1 to 100, but for multiples of three print “Fizz” instead of the number and for multiples of five print “Buzz”. For numbers that are multiples of both three and five, print “FizzBuzz”.

This challenge is a great way to practice your basic programming skills and get comfortable with loops, conditionals, and outputting text to the console. It’s also a good way to start thinking about algorithmic efficiency and optimization, as there are many ways to solve the problem, some of which are more efficient than others.

Here’s an example solution in Python: ```

for i in range(1, 101):
if i % 3 == 0 and i % 5 == 0:
print(“FizzBuzz”)
elif i % 3 == 0:
print(“Fizz”)
elif i % 5 == 0:
print(“Buzz”)
else:
print(i)
```
2. Reverse a String
This challenge is simple but can be a bit tricky if you’re new to programming. The goal is to write a function that takes a string as input and returns the string reversed. For example, if the input is “hello”, the function should return “olleh”. Here’s an example solution in JavaScript: ```

function reverseString(str) {
let reversed = “”;
for (let i = str.length — 1; i >= 0; i — ) {
reversed += str[i];
}
return reversed;
}
```
This solution uses a for loop to iterate through the characters in the string from the end to the beginning and concatenate them to a new string that is returned at the end.
3. Palindrome Checker
A palindrome is a word or phrase that is spelled the same way forwards and backwards, like “racecar” or “level”. The challenge is to write a function that takes a string as input and checks if it is a palindrome. The function should return true if the string is a palindrome and false otherwise.
Here’s an example solution in Python: ```

def is_palindrome(word):
reversed_word = word[::-1]
return word == reversed_word
```

This solution uses Python’s string slicing syntax to reverse the string and then compares the original string to the reversed string to check if they are the same.
4. Find the Longest Word
This challenge is a bit more complex than the previous ones, but it’s still relatively straightforward. The goal is to write a function that takes a string as input and returns the length of the longest word in the string. For example, if the input is “The quick brown fox jumped over the lazy dog”, the function should return 6, since “jumped” is the longest word in the string.

Here’s an example solution in JavaScript:
```
function findLongestWord(str) {
let words = str.split(“ “);
let longest = 0;
for (let i = 0; i < words.length; i++) {
if (words[i].length > longest) {
longest = words[i].length;
}
}
return longest;
}
```
This solution uses JavaScript’s split method to split the string into anarray of words, and then iterates through the array to find the longest word.
5. Sudoku Solver
The final challenge on our list is the Sudoku Solver, which is a more advanced coding challenge that can really test your problem-solving skills. The goal is to write a program that can solve a Sudoku puzzle, given a partially completed puzzle as input. A Sudoku puzzle is a 9x9 grid of numbers, with some of the cells filled in with digits from 1 to 9. The goal is to fill in the empty cells with digits that satisfy the following constraints:

Each row must contain all the digits from 1 to 9. Each column must contain all the digits from 1 to 9. Each of the nine 3x3 sub-grids must contain all the digits from 1 to 9.
Here’s an example solution in Python:

def solve_sudoku(board):
find = find_empty(board)
if not find:
return True
else:
row, col = find
for i in range(1, 10):
if valid(board, i, (row, col)):
board[row][col] = i
if solve_sudoku(board):
return True
board[row][col] = 0
return False
def valid(board, num, pos):
# Check row
for i in range(len(board[0])):
if board[pos[0]][i] == num and pos[1] != i:
return False
# Check column
for i in range(len(board)):
if board[i][pos[1]] == num and pos[0] != i:
return False
# Check sub-grid
box_x = pos[1] // 3
box_y = pos[0] // 3
for i in range(box_y * 3, box_y * 3 + 3):
for j in range(box_x * 3, box_x * 3 + 3):
if board[i][j] == num and (i, j) != pos:
return False
return True
def find_empty(board):
for i in range(len(board)):
for j in range(len(board[0])):
if board[i][j] == 0:
return (i, j)
return None

This solution uses a backtracking algorithm to solve the Sudoku puzzle recursively. The find_empty function finds the first empty cell in the puzzle, and the solve_sudoku function tries to fill in that cell with numbers from 1 to 9 that satisfy the constraints of the puzzle. If a solution is found, the function returns true. Otherwise, it backtracks and tries again with a different number.
